What is a client?

A client is a device or web browser that sends requests to servers to access its information.

What is a server?

A server is a device or a program that sends responds to requests made by other programs or devices called clients.

What is an HTTP request?

An HTTP request is a protocol made from the client requesting data from the server using a URL. Common HTTP request methods are GET, POST, PUT or PATCH and DELETE.

What is an HTTP response?

An HTTP response is the message that the server sends back to the client after making a request. An HTTP response contains a status code, headers and a body.

What is Django and why do we use it?

Django is a 'web framework for perfectionist with deadlines.' It's a set of librares that provides us with code that helps with common tasks required to build a web app.
